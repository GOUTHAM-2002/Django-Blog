Welcome to the Blogger app, developed by Goutham N!

This Blogger app is a testament to my dedication and expertise in Django web development. It incorporates a range of advanced features and functionalities to provide users with a seamless and secure blogging experience.

Key features of the Blogger app include:

  • Authentication and Security: Secure login and registration authentication with password reset functionality via email.
  • User Profile Management: Comprehensive CRUD operations allowing users to manage their profile details, including image upload.
  • Post Management: CRUD operations on user-generated posts with post filterization by username.
  • Pagination: Simple pagination system for smoother navigation through blog posts.
